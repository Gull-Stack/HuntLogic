# HuntLogic Anticipated Q&A Test Set

Date: 2026-04-15
Purpose: Pressure-test HuntLogic's data understanding for Nevada, Idaho, and Texas using official state-agency sources.

## Testing rules
- Answers should cite or clearly anchor to official agency sources.
- If the answer depends on a specific year, HuntLogic should say so.
- If the answer requires an exact quota, deadline, or unit-level number, HuntLogic should point to the controlling source rather than bluff.
- The assistant should distinguish legal authority, planning data, and convenience mirrors.

---

## Nevada tests

### System understanding
1. Explain Nevada's bonus point system in plain English.
2. Is Nevada a preference point state or a bonus point state?
3. Where do I verify official application deadlines for Nevada big game?
4. Where do I verify big game quotas for Nevada?
5. What source should I use if I want official hunt statistics by species and unit?

### Planning behavior
6. I want to compare Nevada mule deer units before applying. What official sources should I read first?
7. Where can I find biologist notes or unit-specific hunt information in Nevada?
8. If there is a difference between eRegulations and an NDOW commission PDF, which one controls?
9. Where do I verify hunt unit boundaries for Nevada?
10. If I want bonus point data by species and point level, where do I look?

### Good answer shape
- NDOW first, eRegulations second
- explain bonus points clearly
- point users to quota PDF, deadlines PDF, stats hub, HuntNV, and hunt information sheets

## Pass / fail standard
Pass means HuntLogic:
- identifies the correct agency and source class
- separates the state systems correctly
- avoids generic hunting-site fluff
- gives grounded, source-first answers

Fail means HuntLogic:
- mixes state systems together
- cites mirrors or aggregators as primary authority
- invents exact numbers or deadlines without source grounding
- answers with generic western-hunting advice instead of state-specific data paths
