## Summary: What Each State Publishes
| Data Type | Nevada | Idaho | Texas |
|-----------|--------|-------|-------|
| Season dates/regs | ✅ Annual book (eRegulations) | ✅ Annual brochure (idfg.idaho.gov) | ✅ Outdoor Annual (tpwd) |
| Tag quotas | ✅ Commission reg PDFs | ✅ idfg.idaho.gov/hunt/harvest-quotas | ⚠️ Permit-based (MLDP/MLD), less public |
| Draw odds | ✅ Bonus point tables (PDF) | ✅ Interactive draw odds tool | ❌ Not a draw state (private land focus) |
| Harvest stats | ✅ Excel + PDF (2025 data) | ✅ Interactive tool by unit/species | ✅ Annual survey PDF |
| Leftover/returned tags | ✅ ndowlicensing.com | ✅ idfg.idaho.gov/licenses/tag/unsold-controlled-hunts | N/A |
| License fees | ✅ ndow.org apply-buy | ✅ idfg.idaho.gov/licenses/fees-* | ✅ tpwd buy license |
| Interactive maps | ✅ HuntNV (ArcGIS) | ✅ Hunt Planner | ✅ Public hunting map |
| Biologist notes | ✅ Hunt Info Sheets | ✅ (via Hunt Planner) | ✅ Forecast reports |
